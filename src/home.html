Embedding Strategies


Behind the scenes, the app is loading six or more different embedding approaches at the same time. In a production app, you might use at most two, and load them on-request.


Classic App

Embed the entire Qlik App

Analytics Chart

Embed a single chart/object

Analytics Chart Data

Extract data from hypercube


Field

Embed a list of field values

Sheet Navigation

List sheets and display a single one

Classic Chart

Embed non-Nebula.js charts

qlik-embed classic/app embeds the full Qlik Sense client in the page.

qlik-embed analytics/chart embeds a chart in the page.

Direct access to analytics application data via existing qlik-embed object.

let chartData = objLayout.qHyperCube.qDataPages[0].qMatrix;

qlik-embed field embeds a list box in the page.

qlik-embed analytics/sheet with custom sheet navigation.
